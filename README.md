# NASM-GUI
Graphical User Interface for NASM compiler. For Windows 10/11.

## Update & Config
To update compiler-... .bat, enter the corresponding path into config-... .txt in the following line:

compiler=path_to_your_file

## ⚠️ WARNING
Do not move config-... .txt.

If the configuration file is moved, the program will be unable to locate the compiler and the helper batch files.

In this case:

Compilation attempts will be marked as "Failed".

The system will trigger errors.
